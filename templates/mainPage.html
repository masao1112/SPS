This project is a prototype of a smart planting system that automates plant care using IoT technology. The motivation behind this project stems from an interest in exploring smart agriculture solutions while contributing to the field with a practical, automated system.

The system features:

  • Watering: Automatically waters plants based on soil moisture levels.
  • Grow Lights: Provides artificial lighting to support plant growth.
  • Visualization: Displays both real-time and historical sensor data using a web interface.
  • User Interaction: Allows users to adjust thresholds for watering and lighting through an interactive interface.

Simulate Circuit Wiring

Using an ESP32 for data transmission and control, along with HTTP for cloud communication, this project integrates Flask, MongoDB, and Firebase for backend processing and data visualization.
